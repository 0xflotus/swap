Swap

Exchange rates library for PHP

View on GitHub Download .zip Download .tar.gz

Swap allows you to retrieve currency exchange rates from various services such as Fixer or Yahoo and optionally cache the results. It is integrated to other libraries like moneyphp/money and provides a Symfony Bundle and a Laravel Package.

QuickStart

$ composer require florianv/swap php-http/message php-http/guzzle6-adapter
use Swap\Builder;

// Build Swap with Fixer.io
$swap = (new Builder())
    ->with('fixer')
    ->build();

// Get the latest EUR/USD rate
$rate = $swap->latest('EUR/USD');

// 1.129
$rate->getValue();

// 2016-08-26
$rate->getDate()->format('Y-m-d');

// Get the EUR/USD rate 15 days ago
$rate = $swap->historical('EUR/USD', (new \DateTime())->modify('-15 days'));
